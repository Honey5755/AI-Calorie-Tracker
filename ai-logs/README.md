# AI Logs

This folder contains the AI-assisted development conversation(s) used to build
**NutriSnap — AI Calorie Tracker** with Claude Code (Anthropic's official CLI).

- `*.jsonl` — raw Claude Code session transcripts (prompts, tool calls, iterations).
- Re-sync the latest logs at any time with:

  ```bash
  bash scripts/sync-ai-logs.sh
  ```

These logs document the prompting and iteration process end-to-end, per the
challenge submission requirements.
